Welcome to Gonçalo’s Personal Notes’s!#

These notes are written using sphinx. For more information, visit the sphinx documentation here. In order to build the notes, just run make clean; make html.

The dependencies to build this documentation are:

  • sphinx;

  • furo (theme);

  • sphinx-multitoc-numbering - check this section for more information;

  • sphinx-copybutton - visit link for more information.
