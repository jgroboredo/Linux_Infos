--------------
Nvidia tearing
--------------

In order to fix tearing, go to ``nvidia-settings``, then select ``X Server Display Configuration``, ``Advanced``
and for all screens select ``Force Composition Pipeline`` and ``Force Full Composition Pipeline``.
